# Profile Image

Place your profile image here as `profile.png`.

**Requirements:**
- File name: `profile.png`
- Recommended size: 800x800px (square)
- Format: PNG, JPG, or WebP
- File size: < 500KB recommended

The image will be displayed in the Hero section with a beautiful rounded frame and glow effect.

If no image is found, a placeholder image from Unsplash will be used as a fallback.
